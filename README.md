# Invenix

Invenix is a custom Retrieval-Augmented Generation (RAG) platform built from scratch. It allows users to upload documents, manage knowledge bases, and chat with their files using local LLMs (Ollama) or free cloud endpoints (Gemini).

## Project Structure
*   `/frontend` - Vite + React (TypeScript) SPA using Tailwind CSS v4.
*   `/backend` - FastAPI application using Python, LangChain, SQLite, and ChromaDB.
